The body's background color is set to "papayawhip" in the CSS. I can read this value (computed to rgba() notation) with JavaScript using window.getComputedStyle(). In this case, I'm grabbing the background property, which returns all of that property's shorthand values, even though only one was explicitly defined in the CSS.

window.getComputedStyle() returns all longhand values for a shorthand property

Above I'm using JavaScript to grab the value of three longhand properties on the first paragraph element on this page: background, animation, and flex. I haven't defined any of these in my CSS, but the return values represent the initial state for all properties.

This demonstrates the kinds of values you should expect to get when using window.getComputedStyle().
